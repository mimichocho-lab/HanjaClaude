## 배경 및 문제

Play 화면의 FlipCard 컴포넌트에 다음 UI 문제가 있다.

1. 카드 폭이 `max-w-xs`(320px)로 고정되어 화면 폭을 충분히 활용하지 못함
2. 뜻(meaning)과 음(pronunciation)이 별도 줄로 표시되어 공간 낭비
3. '탭해서 확인', '탭해서 한자 보기' 안내 텍스트가 카드 내 공간을 차지함
4. 우측 상단 카드 번호가 텍스트 크기 고정(`text-sm`)으로 카드 크기와 무관하게 표시됨

---

## 요구사항

### F-v11-01: 카드 폭 90% 적용

- 카드 컨테이너를 화면 전체 폭의 90%로 변경
- `components/FlipCard.tsx` 수정 대상

### F-v11-02: 뜻음 한 줄 표시

- meaning과 pronunciation을 한 줄로 합쳐 표시
- 형식: `{meaning} {pronunciation}` (예: "물 수")
- 현재: `<p>물</p>` + `<p>수</p>` (2줄)

### F-v11-03: 안내 텍스트 제거

- 앞면: `탭해서 확인` 삭제
- 뒷면: `탭해서 한자 보기` 삭제

### F-v11-04: 카드 번호 폭 비율 적용

- 우측 상단 번호 영역을 카드 폭의 10%로 변경
- `right-4` 고정값 대신 `w-[10%]` 기반 배치 적용
