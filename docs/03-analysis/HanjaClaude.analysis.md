## Gap 목록

### G-01 (Minor) — F-14 뒷면 이미지 미표시

- **파일**: `components/FlipCard.tsx`
- **현황**: 뒷면에 `card.imagePath`가 있어도 이미지 미렌더링, 텍스트만 표시
- **Plan**: F-14 "뜻을 나타내는 그림(이미지) + 아래에 뜻음 텍스트"
- **판정**: Assumptions "텍스트/이모지로 대체 가능"에 의해 허용

### G-02 (Minor) — F-22 삭제 확인 다이얼로그 없음

- **파일**: `app/wrong/page.tsx`
- **현황**: deleteMode에서 X 클릭 시 `removeWrongId()` 즉시 호출 (확인 없음)
- **Plan**: F-22 "삭제 확인 후 제거"
- **영향**: 실수 삭제 가능
- **수정 방안**: `window.confirm()` 또는 인라인 확인 UI
