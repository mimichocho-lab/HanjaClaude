## Gap 목록

### G-01 (Minor) — F-11 뒷면 이미지 미표시

- **파일**: `components/FlipCard.tsx`
- **현황**: 뒷면에 `card.imagePath`가 있어도 이미지 미렌더링, 텍스트만 표시
- **Plan**: F-11 "뜻을 나타내는 그림(이미지) + 아래에 뜻음 텍스트"
- **판정**: Assumptions "텍스트/이모지로 대체 가능"에 의해 허용. 이미지 에셋 추가 시 렌더링 로직 필요.

### G-02 (Minor) — F-19 삭제 확인 다이얼로그 없음

- **파일**: `app/wrong/page.tsx`
- **현황**: deleteMode에서 X 클릭 시 `removeWrongId()` 즉시 호출 (확인 없음)
- **Plan**: F-19 "삭제 확인 후 제거"
- **영향**: 실수로 삭제 가능. UX 경고 없음.
- **수정 방안**: `window.confirm()` 또는 인라인 확인 UI 추가

### G-03 (Minor) — 옵션 화면 선택 카드 수 미표시

- **파일**: `app/options/page.tsx`
- **현황**: 선택된 카드 수 요약("총 N장 선택됨") 표시 없음
- **Plan**: 옵션 화면 "보여지는 특성" — "선택된 카드 수 요약 표시"
- **영향**: 옵션 진입 시 몇 장 선택됐는지 확인 불가
- **수정 방안**: URL query 또는 상태 전달로 옵션 화면에 선택 수 표시
