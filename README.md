# HanjaClaude

구몬한자 B단계 카드
1 한자 순서대로 넘버링
2 양면의 오른쪽 상단에 넘버표시 
3 한면은 한자만(앞면으로 칭함)/다른면은 뜻을 나타내는 그림 + 아래에 뜻음 (뒷면으로 칭함)
4 주요 기능: 한자의 뜻과 음을 숙지하는 연습

## 기본 플레이
   a 처음 초기 홈 화면은 전체 한자가 한 화면에 보임
  (이때는 한자와 뜻음이 모두 같이 보임 -표지 신출한자처럼)
   b 학습할 한자를 눌러 지정함(혹은 전체선택 기능 필요)
   c 지정된 한자만 카드 플레이함
   d 넘버링 순서대로 카드를 한장씩 띄움
     (또는 랜덤 순서대로 할수도 있도록 선택기능 필요)
   e 앞면이 먼저 보일수도 있고 
      뒷면이 먼저 보일수도 있게 선택기능 필요
   f 터치하면 다른 면 전환이 되도록함
   g 옆으로 넘기면 다른 면 전환 없이 다음 카드로 진행

## 추가 플레이 /안 외워진 한자 모아 연습하기
   a  위의 플레이 중 더 연습이 필요한 한자를 선택해 모아두는 
       오답한자 방 만들기
   b  오답 한자방에는 모아진 카드가 한 화면에 띄워짐
   c  선택한 카드를 길게 누르거나 해서 삭제가능 
   d  여기에서 위의 기본 플레이 d~g 플레이 할수 있음
